# Crypto Trading Sim

Unity WebGL game. Start with $10M, pick cryptos, compete.

## Setup

1. Unity 2022.3.62f1
2. Open project
3. Play from `Login.unity`

## Build

```bash
# In Unity: Tools > Build WebGL Now
# Then:
./serve-webgl.sh
```

## Test Users

- Alice: Conservative strategy
- Bob: Risk taker

Both start at $10M.
