Spacing Philosophy
Key Principles Applied:
  1. Edge Anchoring: Push static elements (ID, VIEW) to the far edges
  2. Dollar Buffer: Give monetary values extra breathing room as they can vary in length
  3. Even Distribution: Space middle elements evenly between anchored edges
  4. Visual Hierarchy: Use font size and color to establish importance
  5. No Overlaps: Ensure all text elements have their own clear space
